FAQ about the framework

Donate now
Under what license is Total.js released?
Total.js is released under the free software MIT license. To understand what you can and can't do with the code, please see here.
Can I use different NPM modules in the framework?
Yes, you can use just any module from NPM. The framework doesn't limit you in development of web applications
Does the framework limit me on client-side?
No, Total.js is only server-side (backend) framework and you can use anything on client-side like jQuery, Angular, Embed, React, Backbone, etc..
Which database can be used with the framework?
It is up to you! You can use any type of database such as MongoDB, PostgreSQL, SQLite, MySQL, SQL Server, CouchDB, etc. and the framework will support it.
What are the benefits of the framework?
The framework offers everything a developer needs for web development. It is easy to maintain and update.
Can I rewrite e.g. view engine?
Yes, you can rewrite each functionality of the framework, it's very easy and flexible. Use your favorite view engine, SMTP sender, JavaScript compressor, etc..
Does the framework contain some examples?
The framework contains 100+ examples with full source-code on GitHub. Play with them. Download examples.
What happens to my code before it is sent to the browser?
Your code gets merged, compiled, minified and finally stored in a temporary directory. All requests are subsequently served from this temporary directory.
Did you know? The framework doesn't contain any dependencies. The code size is 690 kB with all comments.
